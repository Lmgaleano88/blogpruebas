COMANDOS BASICOS DE GIT
Git init: Inicia un repositorio de GIT en la carpeta donde es ejecutado el comando Git add: añade los archivos especificados (staging) Git commit -m: Confirma los archivos que están en preparación y los agrega al repositorio
Git commit -am: También guardamos el mensaje para recordar los cambios Git push: Es para enviar nuestros commits a un servidor remoto Git status: Descripción del estado de los archivos
Git rm: (r-filename, -cached)remueve los archivos del index Git config --global user.email: configura un email Git config global user.name: Nombre de usuario como se vera en los commits
config --list: Lista las configuracionesb Git ls:Listar archivos/documentos Git log:Lista los commit que hemos realizado
